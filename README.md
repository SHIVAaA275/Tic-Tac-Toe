A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play on a 3×3 game board with automatic turn switching, winner detection, and draw handling. The game provides a clean and responsive user interface while demonstrating fundamental web development concepts such as DOM manipulation, event handling, functions, arrays, loops, and game logic implementation.

#Features
->Interactive 3×3 game board
->Two-player gameplay (X and O)
->Automatic turn switching
->Winner detection
->Draw game detection
->Responsive and user-friendly design
->Reset/New Game functionality

#Technologies Used
->HTML5
->CSS3
->JavaScript (ES6)

#Learning Outcomes
->DOM Manipulation
->Event Handling
->Conditional Statements
->Arrays and Loops
->JavaScript Functions
->Responsive Web Design
